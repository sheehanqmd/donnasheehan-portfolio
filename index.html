Breath Therapy

JavaScript | CSS3 | React | Redux | React Router | Node.js | Express | PostgreSQL | Authentication

Breath Therapy is a Personal Project. I was given a two week timeline for completion. Breath Therapy is an application that allows consumers to book conferences and workshops. I wad given Award for Best Presentation. Utilized the Eventbrite API and built out custom functionality fit the sites needs. Application allows user to login and store passwords in database with bcrypt using postgreSQL. I built site to be responsive for desktop, tablets and mobile users. The design is straightforward and user friendly. This application allows consumers to book conferences and workshops. github.com/sheehanqmd/zen-breath-therapy-2

Alpha Airlines

JavaScript | CSS3 | React | Redux | React Router | Node.js | Express | PostgreSQL | Authentication

Alpha is an application that enables users to research flight information in real time and book trips, members can save personal information that enables users login and review upcoming trips. My Role on our team of three was to find a usable API and writing to bring in flight data to render in application. We used Sabre OAuth API to generate real time flight information. Worked with team to develop several algorithms for the implementation of incoming data from API. Wrote tests for components via Jest test/Cypress test. github.com/flight-club/flight-club
